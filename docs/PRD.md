# PRD - 지원나침반 AI

## Product Statement

지원나침반 AI는 사용자의 실제 사업 자료와 지원사업 공고 원문을 근거로 지원 자격, 적합도, 준비 부족 항목을 분석하고 PSST 사업계획서 초안과 제출 체크리스트를 만드는 한국어 AI 워크스페이스다.

## Primary User

- 음악·악기 교육 웹앱을 혼자 개발하는 1인 창업자
- 향후 타 업종 예비·초기 창업자로 확장

## MVP Flow

1. 로그인
2. 사업 프로필과 보유 자산 확인
3. 공고문 텍스트 또는 파일 등록
4. 공고 핵심 정보 구조화와 근거 위치 확인
5. 자격 판정: 지원 가능, 지원 불가, 확인 필요
6. 적합도 점수와 부족 항목 확인
7. 지원 프로젝트 생성
8. PSST 문항별 초안 생성
9. 근거 없는 문장과 미확인 사실 표시
10. 제출 체크리스트, 예상 질문, Markdown/DOCX/PDF export

## Non-Goals

- 정부 사이트 자동 로그인 또는 자동 제출
- 공동인증서, 간편인증, 전자서명 자동화
- K-Startup 등 외부 사이트 무단 크롤링
- 선정 확률 백분율 단정
- 법률, 세무, 노무 확정 자문
- 사용자 승인 없는 실적과 수치 생성
- HWP/HWPX 본문 직접 파싱
- 비용 제한 없는 멀티에이전트 반복

## Success Criteria

- 사용자가 15분 안에 공고 등록부터 초안 생성까지 도달한다.
- 모든 핵심 결론은 citation 또는 `unknown` 상태를 가진다.
- Hard Gate 실패가 있는 경우 `eligible`로 표시되지 않는다.
- Claim Linter가 출처 없는 정량 주장과 최상급 표현을 차단한다.
- 모든 필수 검증 명령이 통과한다.
